## Python MySQL Academic Task System

Academic project developed in Python with MySQL integration, focused on user authentication and task management for a gamified college system.
The system includes different access levels for professors and students, allowing task creation, visualization and deletion based on user permissions and course association.

## Features

- User authentication system
- Role-based access control (Professor / Student)
- Task creation and deletion
- Task visualization by course
- MySQL database integration
- Relational database queries using SQL

## Technologies

- Python
- MySQL
- SQL
- mysql-connector-python

##Concepts Practiced

- Backend development
- Database integration
- CRUD operations
- Authentication systems
- Relational database management
- User permission handling

## Objective

Practice software development and database concepts through the creation of a functional academic management system integrated with a game project.
